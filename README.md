# Simple string encryption written in lua

## Examples

### Encryption

> >library:encrypt("Hello World!")
OUTPUT: ::14.4::20.2::21.6::21.6::22.2::6.4::17.4::22.2::22.8::21.6::20::6.6::

### Decryption
> >library:decrypt("::14.4::20.2::21.6::21.6::22.2::6.4::17.4::22.2::22.8::21.6::20::6.6::")
OUTPUT: Hello World!
